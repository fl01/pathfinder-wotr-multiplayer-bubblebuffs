# [BubbleBuffs](https://github.com/factubsio/BubbleBuffs) multiplayer compatibility Mod for Pathfinder: Wrath of the Righteous

Basic example how to add multiplayer support for an existing mod.

- Buff usage is synced for everyone
- Buff config is synced from initiator

[More details](/docs/details.md)

## How to
1. Install [BubbleBuffs](https://github.com/factubsio/BubbleBuffs)
2. Install [Multiplayer](https://github.com/fl01/pathfinder-wotr-multiplayer)
3. Download latest from releases
3. Install via `UnityModManager`
   - or by manually extracting `.zip` file content into `Pathfinder Second Adventure\Mods`, but this requires `UnityModManager` to be installed for the game already
